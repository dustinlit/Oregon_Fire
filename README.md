*Author*: Dustin Littlefield\

## **Objective:** 
A geographic and statistical analysis of variables that contribute to wildfire risk and severity in Oregon. Construct various machine language models (Trees, Neural Networks, Regression) to identify which factors reltate to large and damaging fires.
